## 📝 Usage Guidelines

### Tone
- **Professional security tool** — not hype, not marketing fluff
- **Confident, data-driven** — back claims with numbers
- **Technical but accessible** — explain concepts clearly

### Do's
✅ Lead with the problem (visibility gap)  
✅ Use the "Shodan for AI Agents" analogy  
✅ Highlight 95%+ confidence, 27 techniques, OWASP 20/20  
✅ Emphasize open-source, self-hostable, 5-minute deploy  
✅ Share real detection examples  

### Don'ts
❌ Overpromise on cloud scanner timeline  
❌ Claim 100% detection (nothing is perfect)  
❌ Use fear-mongering ("AI agents will destroy you!")  
❌ Ignore false positives (acknowledge and improve)  
❌ Engage with bad-faith criticism
