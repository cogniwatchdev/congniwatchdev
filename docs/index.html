✨ Key Features

🔎 Multi-Layer Detection

HTTP headers, API endpoints, WebSocket protocols, TLS/JA3 fingerprinting, and behavioral analysis for accurate framework identification.

🎯 23+ Frameworks

OpenClaw (all variants), Agent-Zero, LangGraph, AutoGen, CrewAI, Microsoft/Google frameworks, and more with confidence scoring.

📊 Real-Time Telemetry

Monitor agent connections, API calls, WebSocket messages, response times, and error rates with historical trend analysis.

🛡️ OWASP Compliant

Full compliance with OWASP Top 10:2025 and LLM Top 10:2025, JWT authentication, RBAC, and AES-256-GCM encryption.

🐳 Docker Ready

Deploy in minutes with Docker Compose. Pre-configured security, isolated environment, and easy updates.

🔌 REST API

Complete programmatic control with RESTful API, WebSocket real-time updates, and SDK support for Python and JavaScript.
