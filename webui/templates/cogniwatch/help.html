Getting Started Guide

Everything you need to start detecting AI agents

Quick Start (5 Minutes)

1

Access the Dashboard

Navigate to your CogniWatch instance at http://localhost:8080

2

Configure Your First Scan

Go to Scan → Enter your network range (e.g., 192.168.1.0/24) → Select "Standard Scan" → Click "Start Scan"

3

View Results

Watch real-time progress on the Dashboard. Detected agents will appear in the discovery table with confidence scores.

4

Review Security

Check the Security tab for any flagged issues. Address critical alerts first.

API Documentation

CogniWatch provides a RESTful API for automation and integration with your existing tools.

Example API Calls

# Start a network scan

curl -X POST http://localhost:8080/api/scan/start \
-H "Authorization: Bearer YOUR_TOKEN" \
-d '{"network": "192.168.1.0/24", "profile": "standard"}'

# Get detected agents

curl http://localhost:8080/api/agents \
-H "Authorization: Bearer YOUR_TOKEN"

# Get security alerts

curl http://localhost:8080/api/security/alerts \
-H "Authorization: Bearer YOUR_TOKEN"
