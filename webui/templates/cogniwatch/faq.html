Frequently Asked Questions

Everything you need to know about CogniWatch

General

What is CogniWatch?

CogniWatch is an AI agent detection network that scans your network to identify AI agents and frameworks. Think of it as "Shodan for AI Agents" - we help you discover what AI systems are running on your network, assess their security posture, and monitor for unauthorized deployments.

How does CogniWatch work?

CogniWatch uses advanced fingerprinting techniques to detect AI agents by analyzing:

  • Network port signatures and service banners
  • HTTP headers and API response patterns
  • Framework-specific endpoints and behaviors
  • Telemetry data and communication patterns

What frameworks can you detect?

We currently detect 23+ AI frameworks including:

LangChain AutoGen CrewAI Agent Zero LlamaIndex Haystack Semantic Kernel OpenClaw And more...

Technical

How accurate is detection?

Our confidence scoring system provides:

  • Confirmed (≥85%) - High-confidence detection with multiple matching signatures
  • Likely (60-84%) - Strong indicators present, manual verification recommended
  • Possible (30-59%) - Some matching patterns, could be false positive
  • Unknown (<30%) - Weak signals, likely not an AI agent

Does scanning affect network performance?

CogniWatch uses non-intrusive scanning techniques. A typical scan of a /24 subnet (254 hosts) completes in 2-5 minutes with minimal network impact. You can configure scan intensity and concurrent threads to balance speed vs. network load.

Can I schedule automatic scans?

Yes! You can schedule recurring scans with custom intervals (hourly, daily, weekly). Scheduled scans run in the background and results are automatically added to your dashboard.

Security & Privacy

Is CogniWatch secure?

Absolutely. CogniWatch runs entirely on your infrastructure - we don't send any data to external servers. All scan data, agent information, and telemetry stay within your network. We also provide security recommendations for any agents we find with misconfigurations.

Do you store my data?

No. CogniWatch is self-hosted and all data is stored locally. You have complete control over retention policies and can export or delete data at any time.

Can detected agents be exploited?

CogniWatch only identifies agents - it doesn't attempt to exploit or penetrate them. However, we do flag security issues like exposed APIs, missing authentication, or default configurations so you can remediate them.
